Miscellanea¶

class pygmo.thread_safety¶

Thread safety level.

This enum defines a set of values that can be used to specify the thread safety of problems, algorithms, etc.

basic = pygmo.core._thread_safety.basic¶

Basic thread safety – any concurrent operation on distinct instances is safe

none = pygmo.core._thread_safety.none¶

No thread safety – any concurrent operation on distinct instances is unsafe

class pygmo.evolve_status¶

Evolution status.

This enumeration contains status flags used to represent the current status of asynchronous evolution/optimisation in pygmo.island and pygmo.archipelago.

See also

pygmo.island.status and pygmo.archipelago.status.

idle = pygmo.core._evolve_status.idle¶

Idle – no asynchronous operations are ongoing, and no error was generated by an asynchronous operation in the past (value = 0)

busy = pygmo.core._evolve_status.busy¶

Busy – asynchronous operations are ongoing, and no error was generated by an asynchronous operation in the past (value = 1)

idle_error = pygmo.core._evolve_status.idle_error¶

Idle with error – no asynchronous operations are ongoing, but an error was generated by an asynchronous operation in the past (value = 2)

busy_error = pygmo.core._evolve_status.busy_error¶

Busy with error – asynchronous operations are ongoing, and an error was generated by an asynchronous operation in the past (value = 3)
